# Rtl
This gem helps you in checking whether a language is written from right-to-left (RTL) or vise versa.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rtl'
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install rtl
```
